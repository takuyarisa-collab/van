# Static Phaser Prototype

This prototype is now a single static page with no build step and no npm dependencies.

## Run

Open `index.html` directly in a browser, or serve this folder with any static server.

## Controls and loop

- One-thumb movement with slight inertia (touch or mouse hold/drag).
- Enemies spawn and chase.
- Auto-fire targets nearest enemy.
- XP pickups level you up and improve fire rate.
- HUD shows HP/Level/XP.
- On defeat, the run auto-restarts.
